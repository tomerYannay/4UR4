# Skills

**No skills exist yet — by design.**

Per [GOV-012 Skill Creation Threshold](../governance/skills-policy.md), a shared
skill is created **only** when a procedure is used by **≥ 2 tickets or ≥ 2 agents**.
No procedure has yet crossed that threshold, so this directory holds only the
policy pointer below. Creating a skill "just in case" is bloat and is not allowed.

## When a skill *is* justified

A skill file (added here by Product Steward approval) must record:

- **Purpose** — what it does.
- **Inputs / Outputs** — its contract.
- **Consumers** — the ≥ 2 tickets/agents that justify it (required).
- **Owner** — accountable agent.

The Auditor retires skills that fall below the ≥ 2-consumer threshold.

See [`governance/skills-policy.md`](../governance/skills-policy.md).
